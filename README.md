# 🎌 AnimeZone — მულტიენოვანი ანიმე პლატფორმა

AnimeZone არის თანამედროვე ვებ-აპლიკაცია, რომელიც შექმნილია **Flask** ფრეიმვორქის გამოყენებით. საიტი მომხმარებელს სთავაზობს სრულყოფილ მუქ ვიზუალს სტაფილოსფერი აქცენტებით (Dark & Orange Theme) და სამენოვან მხარდაჭერას.

---

## ✨ ძირითადი ფუნქციონალი (Features)

* **🌐 სრული მულტიენოვნება (Multi-language Support):** საიტის ენის შეცვლა შესაძლებელია მომენტალურად 3 ენაზე: **ინგლისური (EN), ქართული (KA)** და **იაპონური (JA)**. ბრაუზერი ავტომატურად იმახსოვრებს მომხმარებლის არჩევანს `session`-ების დახმარებით.
* **🎨 Dark & Orange დიზაინი:** ყველა სტილი სრულად არის გამოცალკევებული `static/css/style.css` ფაილში. საიტს აქვს მორგებული ანიმაციები (Hover Effects) ანიმეს ბარათებზე.
* **🔐 ავტორიზაცია და როლები:** სისტემაში ინტეგრირებულია `Flask-Login`. სტუმრებს შეუძლიათ მხოლოდ დათვალიერება, ხოლო `Admin` როლის მქონე მომხმარებელს აქვს ანიმეების დამატების, რედაქტირებისა და წაშლის (CRUD) უფლება.
* **🗄️ ავტომატური მონაცემთა ბაზა:** `init_db.py` სკრიპტის გაშვებისას ბაზაში ავტომატურად იწერება პოპულარული ანიმეები (Jujutsu Kaisen, One Piece, JoJo და ა.შ.) თავისი სამენოვანი აღწერებით.

---

## 🛠️ ტექნოლოგიური სტეკი (Built With)

* **Backend:** Python / Flask
* **Database ORM:** Flask-SQLAlchemy (SQLite)
* **Authentication:** Flask-Login / Werkzeug Security
* **Forms:** Flask-WTF / WTForms
* **Frontend:** HTML5, CSS3 (Custom Styles), Bootstrap 5
